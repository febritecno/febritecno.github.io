Yuk Kenalan Dulu Sama Framework Octopress

Nov 16th, 2014 2:50 pm | Comments

Sebelum saya mengenalkan jauh tentang Framework ini. Mari kita bersama-sama mengenal dulu apa sih octopress ?? :). saya yakin yang belum pernah mengenal pasti bertanya-tanya octopress itu apa dan dsb.yuk kita kenalan dulu.

Apa itu Octopress ?

Octopress adalah Jekyll blogging yang terbaik dan responsif.

Keunggulan Octopress : - Octopress sports a clean responsive theme written in semantic HTML5, focused on readability and friendliness toward mobile devices. - Code blogging is easy and beautiful. Embed code (with Solarized styling) in your posts from gists, jsFiddle or from your filesystem. - Third party integration is simple with built-in support for Pinboard, Delicious, GitHub Repositories, Disqus Comments and Google Analytics. - It’s easy to use. A collection of rake tasks simplifies development and makes deploying a cinch. - Ships with great plug-ins some original and others from the Jekyll community â€” tested and improved.

Catatan : Octopress memerlukan versi Ruby minimal 1.9.3 - p0

Menurut saya octopress merupakan Framework yang mudah untuk dipelajari dan portable bisa disimpan di Flashdiks. untuk penginstallannya tidak membuntuhkan hal yang sulit, tinggal lihat tutor selesailah sudah !! paling enak tidak menggunakan Xammp dan tidak memiliki database.

Saya lihat bahwa pengguna octopress masih sangatlah sedikit dan kurang diminati !! padahal menggunakannya sangat mudah dan efisian sekali.
