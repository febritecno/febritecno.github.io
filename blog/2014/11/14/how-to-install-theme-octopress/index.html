How to Install Theme Octopress

Nov 14th, 2014 10:17 pm | Comments

Hello Guys, saya mau coba aja membuat hal yang berbeda dengan octo kamu. semua orang pasti senang dengan tampilan yang berbeda pada setiap laman blog maupun website.

Oke, kita fokus ke tujuan kita, kita akan mencoba untuk meninstall theme berbeda pada octopress kita.

Eh, ngomong-ngomong kalian pakai ? jommla,druppal atau wordpress. kalian terlalu mainstream banget. coba kalian gunakan Octopress, lebih responsif dan portable.

Kanapa bisa portable? karena Octapress tidak mengunakan hostingan,php,database. dan lebih mudah untuk mengatur postingan.

Serta yang paling hebat bebas dari hacker, yaiyalah gk databasenya dan hostingan di flashdisk/HDD cuma menggunakan media github/Social code saja untuk bisa tampil diinternet. oke gitu aja pengenalannya.

Berikut cara menginstall/mengganti theme pada Octopress :
  1. Buka git-scm/Git bash #/sudah diinstall semua komponen Framework Octapress
  2. Ketik Directory Octopress kalian :

    $ cd Documents/Github/[Forder Octo]/[forder kalian]/

  3. Kalau sudah selesai pidah directory langsung teruskan langkah dengan mengetik kode ini :

    $ git submodule add GIT_URL .themes/THEME_NAME

  4. Lanjut ketik ini untuk install theme Octapress:

    $ rake install[‘THEME_NAME’]

  5. Klau udah selesai bisa di Generate dlu

    $ rake generate

  6. Terus dihostingkan di Github

    $ Rake deploy

  7. Selesai, coba lihat !!


Dari disilah ilmu kamu bertambah, dan semoga bermanfaat bagi kalian semua.

Note : download theme disini, copy link dan nama theme taruh pda git-scm!! # Theme Kesini
